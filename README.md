# Polski dla trenera

Тренажёр польского языка с нуля до A2 для тренера по теннису: 12 уроков (~445 слов), 8 диалогов, интервальные повторения, тесты 5 типов, аудирование, числа на слух, грамматика и говорение с проверкой через микрофон (Web Speech API). Работает как PWA — можно добавить на главный экран.

Открыть: https://julieholubeva-dot.github.io/polski-trainer/

На iPhone открывать в Safari (в Chrome для iOS не работает распознавание речи).
