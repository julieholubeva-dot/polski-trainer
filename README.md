# Polski dla trenera

Тренажёр польского языка с нуля до A2 для тренера по теннису: 12 уроков, ~445 слов и фраз, тесты, аудирование, сборка фраз и говорение с проверкой через микрофон (Web Speech API).

Открыть: https://julieholubeva-dot.github.io/polski-trainer/

На iPhone открывать в Safari (в Chrome для iOS не работает распознавание речи).
